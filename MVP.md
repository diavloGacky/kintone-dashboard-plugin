# MVP仕様書 - kintone Dashboard Plugin

## 概要

kintoneの一覧画面にダッシュボードを表示するプラグイン。
APIトークンを使用してkintone APIと連携し、ノーコードでダッシュボードを構築できる。

---

## アーキテクチャ

```
[プラグイン設定画面]
  └── APIトークン入力
  └── ダッシュボード設定（ウィジェット追加・配置・設定）
  └── 設定を保存（kintoneプラグイン設定として保存）

[kintone一覧画面]
  └── プラグインがダッシュボードを差し込み表示
  └── APIトークンを使用してkintone REST APIでデータ取得
  └── 設定に基づいてウィジェットを描画
```

---

## MVP機能一覧

### 1. 数値カード

- kintoneアプリの指定フィールドの合計・件数・平均を表示
- 表示項目：タイトル・数値・単位
- 対応集計：SUM / COUNT / AVG

### 2. テーブル（一覧表示）

- kintoneアプリのレコードを表形式で表示
- 表示フィールドをプラグイン設定で選択
- 最大表示件数：設定可能（デフォルト20件）

### 3. 棒グラフ

- kintoneアプリのデータを棒グラフで表示
- X軸：指定フィールド（文字列系）
- Y軸：指定フィールド（数値系）の集計値

### 4. 円グラフ

- kintoneアプリのデータを円グラフで表示
- 分類フィールドと数値フィールドを指定
- 凡例表示あり

### 5. フィルタ

- 日付範囲・ドロップダウン値・テキストでレコードを絞り込み
- フィルタ変更時にウィジェット全体を再描画

### 6. ドラッグ＆ドロップ配置

- ウィジェットをドラッグ＆ドロップで自由に配置変更
- グリッドレイアウトベース

### 7. レイアウト保存

- ウィジェットの配置・サイズをプラグイン設定として保存
- 次回アクセス時に復元

### 8. ノーコード設定

- プラグイン設定画面でウィジェットを追加・削除・設定
- コードを書かずにダッシュボードを構築可能

## 画面仕様

### プラグイン設定画面

| 項目 | 内容 |
|------|------|
| APIトークン | kintone APIトークン（テキスト入力） |
| 対象アプリID | データ取得元のkintoneアプリID |
| ウィジェット一覧 | 追加済みウィジェットの一覧・編集・削除 |
| ウィジェット追加 | 種別選択→各種設定→追加 |
| プレビュー | （将来対応） |

### kintone一覧画面

| 項目 | 内容 |
|------|------|
| 表示エリア | 一覧画面上部にダッシュボードを差し込み |
| ウィジェット表示 | 設定に基づいてグリッドレイアウトで描画 |
| フィルタUI | フィルタ設定時はフィルタバーを表示 |
| 編集モード | ドラッグ＆ドロップ切り替えボタン |

---

## API連携仕様

- **認証方式**: APIトークン（`X-Cybozu-API-Token` ヘッダー）
- **使用API**:
  - レコード取得: `GET /k/v1/records.json`
  - フィールド情報取得: `GET /k/v1/app/form/fields.json`
- **CORS対応**: kintoneプラグインはkintoneドメイン内で動作するためCORS問題なし

- Excel出力
- 予測分析
