# kintone-dashboard-plugin

## 概要

kintone向けのダッシュボード作成プラグイン。

kintone標準グラフよりも見やすく、
現場担当者や管理者が簡単に状況を確認できる
ダッシュボードを作成できるようにする。

## 解決したい課題

- kintone標準グラフが使いにくい
- 複数の情報を1画面で見たい
- Excelに出力して集計する作業を減らしたい
- 現場向けの見やすい画面を作りたい

## MVP機能

- 数値カード表示
- 一覧テーブル表示
- 棒グラフ表示
- 設定画面
- kintoneアプリデータ取得

## 想定ユーザー

- kintone管理者
- 情シス担当者
- 業務改善担当者
- 中小企業の現場リーダー

## 開発方針

- まずは小さく作る
- 使いやすさを重視する
- 販売できる品質を目指す
- Claude Codeを開発AIとして活用する
