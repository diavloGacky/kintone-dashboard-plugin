## プロダクト概要

kintoneの一覧画面にPower BI風ダッシュボードを表示するプラグイン。
プラグイン設定画面（ビジュアルビルダー）でノーコードにてウィジェットを配置・設定できる。

## アーキテクチャ

### 設定画面（config）- Power BI風 3パネルビルダー
```
[左パネル] ウィジェットパレット（2列グリッド）
  └── データウィジェット：数値カード・テーブル・棒グラフ・円グラフ・フィルタ
  └── 装飾ウィジェット：テキスト・図形（プリセット10種をドラッグ&ドロップ）

[中央] キャンバス（gridstack.js）
  └── ウィジェットを自由配置・リサイズ
  └── データを読み込むとリアルタイムプレビュー表示

[右パネル] プロパティパネル
  └── ウィジェットクリックで設定を表示
  └── 変更即時反映
```

設定は `kintone.plugin.app.setConfig()` で保存。

### ダッシュボード表示（desktop.js）
```
app.record.index.show イベント
  └── プラグイン設定読み込み
  └── 対象ビューのみ表示（targetView設定）
  └── APIトークン + アプリIDでレコード取得
  └── gridstack.jsで自由配置描画
  └── フィルタ → 絞り込みAPIを叩いて即時再描画
  └── 編集モード → ドラッグ&ドロップでレイアウト変更・保存
```

## ウィジェット一覧

| 種別 | 説明 | 設定項目 |
|------|------|----------|
| 数値カード | 集計値を大きく表示 | フィールド・集計方法・単位・タイトル表示 |
| テーブル | レコード一覧 | 表示フィールド・最大件数・タイトル表示 |
| 棒グラフ | X/Y軸グラフ | X軸・Y軸・集計方法・タイトル表示 |
| 円グラフ | 割合グラフ | 分類・数値・集計方法・タイトル表示 |
| フィルタ | 絞り込み | 種別（日付範囲/ドロップダウン/テキスト）・フィールド |
| テキスト | 自由テキスト | キャンバス上で直接入力・フォント・色・背景・揃え |
| 図形 | 装飾図形 | 10種プリセット（四角・円・ライン）・色・透明度・角丸 |

## 重要な仕様・注意点

### z-index
- 装飾ウィジェット（テキスト・図形）: z-index:1（背面）
- データウィジェット: z-index:2（前面）
→ 図形をデータウィジェットの背景として使える

### フィルタ動作
- フィルタ変更 → APIで絞り込み → データウィジェットのみ再描画
- フィルタUIは再描画しない（入力値が消えるバグ防止）

### テキストボックス
- キャンバス上で直接タイピング可能
- `⚙` ボタンでスタイル設定を開く

### タイトル表示
- 各データウィジェットは「タイトル表示/非表示」を個別設定可能

### ビュー指定
- 設定画面で特定のビュー（タブ）のみに表示を限定できる
- 「グラフ」タブや専用ダッシュボードタブに表示するのが推奨

### テーブルのフィールドラベル
- 設定時にフィールドコード→ラベルのマッピングを `fieldLabels` として保存
- 表示時はラベル名で列ヘッダーを表示
